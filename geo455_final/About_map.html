Stargazing Site Finder

GEO 455 — Web Mapping Final Project

About this map

This interactive map helps users find ideal stargazing locations by identifying areas with low light pollution.

How to use the map

Explore layers such as protected areas and nighttime lights to locate darker skies.

Data Sources

  1. National Parks Boundaries
  2. Urban Areas Dataset — U.S. Census Bureau TIGER/Line Files (accessed 2026). Used to represent urban development patterns.
  3. Weather alerts
  4. VIIRS Nighttime Lights — NASA EarthData (accessed 2026). Used to visualize light pollution intensity.

Navigation

Basemaps

  1. Nighttime: used to understand the light pollution in urban areas
  2. streemap: used to visualize the pathway to reach the sites
  3. Imagery

Project Info

Methods

  • Downloaded and cleaned datasets from federal GIS data sources.
  • Clipped and simplified layers to improve web map performance.
  • Converted datasets into web-friendly formats for easy display using ArcGIS pro.
  • Built the interactive map using HTML, CSS, JavaScript, and Leaflet.
  • Added interactive controls including search, layer toggles, and popups.
  • Published the final project on github.

Limitations and Credits

Nighttime light data represents general light intensity and does not reflect temporary conditions such as weather or local lighting changes. Some national parks may also have limited nighttime public access.
