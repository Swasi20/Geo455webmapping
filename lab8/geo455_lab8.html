Lab 8: Commute Flow Map to Dane County (2011-2015) GEO 455: Webmapping
Map Info

This map displays commute flow patterns to Dane County from 2011-2015. The animated flow map shows number of commuters from neighboring counties. The map uses leaflet.migrationLayer to create the flow animation. The data is from the US Census Bureau's Longitudinal Employer-Household Dynamics (LEHD) program, which provides detailed information on employment and commuting patterns in the United States.
